# BlogPostAPI
- To Start the app, please type the following in command line with the same directory:

[ *** option to create virtual env ***]
virtualenv appenv       // to create work env
source appenv/bin/activate      // to activate it
--------------------------
pip3 install -r requirements.txt      // install depdencies for the app
python3 app.py     //to start the app

- app will run on:  http://127.0.0.1:5000
